PROJECT
BRANDING

Hiervoor heb ik een logo gemaakt en een font gekozen. Ik heb dit gedaan in Figma en in Adobe illustrator. Ik heb veel logo's gemaakt om te kijken wat het beste paste bij de opdracht. Uiteindelijk heb ik een logo gekozen en daar een kleurenpallet bij gemaakt. Daarop heb ik feedback gevraagd aan medestudenten en docenten. Ik heb ook een poster gemaakt voor de branding. Dit heb ik gedaan in Adobe illustrator.

Die poster heb ik toen opgehangen tussen de posters van andere studenten uit de klas. Hieruit kreeg ik feedback van de stakeholder en de docenten. Daarop heb ik de poster aangepast en opnieuw feedback gevraagd. Ook heb ik een huisstijl gemaakt voor het project. Dit heb ik gedaan in figma.
