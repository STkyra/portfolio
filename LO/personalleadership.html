PORTFOLIO

Voor mijn portfolio heb ik feedback gevraagd aan docenten, dit heb ik in feedpulse gezet. Als ik was vergeten te vragen om een feedpulse dan zette ik de feedback in figma zelf. Zo had ik snel itteraties en had ik ook nog alle feedback bij me. Ik vond het heel leuk om aan mijn portfolio te werken, ook heb ik een aantal dingen geleerd hiermee. Zo kan ik nu makkelijk werken met Git en weet ik nu ook hoe ik het beste grid en flexbox kan gebruiken.

Ik heb ook geleerd dat het beter is om Gitlab te gebruiken dan Github. Dit omdat Gitlab een backup heeft vanuit school zelf. Zo kan school altijd bij mijn oude portfolio's en projecten al heb ik het verwijderd. Dankzij Maikel weet ik nu ook hoe ik mijn Github repository over kan zetten naar Gitlab.
