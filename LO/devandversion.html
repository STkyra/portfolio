PORTFOLIO

Voor mijn portfolio heb ik html, css en javascript gebruikt. Ik heb een carousel gemaakt met javascript. Hierin kan je klikken op de leeruitkomsten en dan ga je naar de pagina voor die leeruitkomst. Ook heb ik een stikky navbar, dit heb ik gedaan met javascript. De javascript van de sticky nav zit in een apparte js file om dit overzichtelijker te maken. Verder heb ik nog drie css files, een voor de leeruitkomst pagina's, een voor de index pagina en een voor de afbeeldingen. De afbeeldingen css file heb ik puur zodat ik een gallerij kan maken, dit omdat alle afbeeldingen andere grootes zijn. Zo kan ik makkelijk alles aanpassen zonder te veel te zoeken.

Nattuurlijk heb ik een git voor mijn portfolio aangemaakt. Ik maak vaak grote commits, vandaar dat er minder zijn. Verder heb ik ook in de css een smooth scroll toegevoegd zodat het beter scrollt. Ook voeg ik comments toe aan de code zodat het duidelijk is wat wat is. Dit ook zodat als iemand anders de code zou lezen, dat diegene het ook snapt.

PROJECT
DEVELOPMENT

Dit project was een vervolg op het media campaign project. Ik heb de site gemaakt met de informatie die we hadden verzameld van de studenten. Hiervoor heb ik html, css en een beetje javascript gebruikt. Ik en Loes hebben feedback gevraagd aan maikel voor de easter egg, hij heeft ons een beetje geholpen hiermee met ideeen. Voor de javascript heb ik een beetje hulp gekregen van een medestudent, en html en css heb ik zelf gedaan.

Ook heb ik een smooth scroll toegevoegd aan de site zodat het er wat beter uitziet en over loopt. Ik heb gekeken naar de feedback van de stakeholder en studenten en heb de site aangepast waar nodig. In de code heb ik comments toegevoegd zodat het duidelijk is wat wat is. Dit ook zodat als iemand anders de code zou lezen, dat diegene het ook snapt. Loes heeft ook een gitlab repository aangemaakt en daar heb ik de code in gezet.
