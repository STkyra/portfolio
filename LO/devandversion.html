PROJECT
DEVELOPMENT

Dit project was een vervolg op het media campaign project. Ik heb de site gemaakt met de informatie die we hadden verzameld van de studenten. Hiervoor heb ik html, css en een beetje javascript gebruikt. Ik en Loes hebben feedback gevraagd aan maikel voor de easter egg, hij heeft ons een beetje geholpen hiermee met ideeen. Voor de javascript heb ik een beetje hulp gekregen van een medestudent, en html en css heb ik zelf gedaan.

Ook heb ik een smooth scroll toegevoegd aan de site zodat het er wat beter uitziet en over loopt. Ik heb gekeken naar de feedback van de stakeholder en studenten en heb de site aangepast waar nodig. In de code heb ik comments toegevoegd zodat het duidelijk is wat wat is. Dit ook zodat als iemand anders de code zou lezen, dat diegene het ook snapt. Loes heeft ook een gitlab repository aangemaakt en daar heb ik de code in gezet.
