PORTFOLIO

For my portfolio, I used HTML, CSS, and JavaScript. I created a carousel with JavaScript, where you can click on the learning outcomes and be taken to the page for that outcome. I also made a sticky navbar using JavaScript. The JavaScript for the sticky nav is in a separate JS file to keep things organized.

Additionally, I have three CSS files: one for the learning outcome pages, one for the index page, and one for the images. The images CSS file is specifically for creating a gallery since all the images have different sizes. This makes it easy to adjust everything without too much hassle.

Of course, I created a Git repository for my portfolio. I tend to make large commits, so there are fewer commits overall. In the CSS, I also added smooth scrolling for a better scrolling experience. I include comments in the code to make it clear what each part does, so that anyone else reading the code can understand it too.
